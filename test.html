Local Network Access · conformance harness

LNA Test Platform

Drive every gated surface from one page and watch the browser's behavior in real time. This page must be loaded from a public HTTPS origin (e.g. the deployed site) for the checks to be meaningful — opened from localhost, LNA never fires because the page is already loopback.

Before you start

  1. Enable enforcement: chrome://flags/#local-network-access-check → Enabled (Blocking) (Chrome 138–141). Chrome 142+ enforces by default.
  2. Run the local test daemon so the probes have something to hit:
    curl -sO https://raw.githubusercontent.com/damionrashford/lna/main/test-server.ts && bun test-server.ts → serves 127.0.0.1:7966 with CORS + a WebSocket echo.
  3. Run Run all below, or fire tests individually. Grant the permission prompt when it appears to see verdicts flip.
